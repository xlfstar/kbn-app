# kbn-electron-app

An Electron application with Vue

## Recommended IDE Setup

- [VSCode](https://code.visualstudio.com/) + [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint) + [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar)

## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```

企业/ERP系统：
Procurement Management - 全面采购管理（包含战略、流程、供应商等）

Purchase Management - 采购过程管理

具体功能模块：
采购订单管理: Purchase Order Management

供应商管理: Supplier Management / Vendor Management

采购申请: Purchase Requisition

采购流程: Procurement Process / Purchasing Process

采购分析: Procurement Analytics

专业术语：
战略采购: Strategic Procurement

集中采购: Centralized Procurement

分散采购: Decentralized Procurement

电子采购: E-Procurement

采购周期: Procurement Cycle

部门/职位名称：
采购管理部门: Procurement Management Department

采购经理: Procurement Manager / Purchasing Manager

采购专员: Procurement Specialist

相关系统名称：
采购管理系统: Procurement Management System (PMS)

供应链管理: Supply Chain Management (SCM)

企业资源计划: Enterprise Resource Planning (ERP)
